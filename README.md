# Help UFBA

Ferramentas acadêmicas para estudantes do **Universidade Federal da Bahia**.

🔗 **Site:** [rodrigogduca.github.io/help-ufba](https://rodrigogduca.github.io/help-ufba/)

## Ferramentas

- **Passei UFBA?** — Calcule sua média semestral e descubra a nota mínima para a final.
- **Calculadora de Faltas** — Descubra quantas faltas você ainda pode ter com base na carga horária da disciplina (presença mínima: 75%).
